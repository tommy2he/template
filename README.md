# template

* code template
* development environment configuration template

provide the simplest templates of various development environments that have been set up .

If you are a full-stack developer, you can utilize these templates to swiftly establish a vscode development environment tailored to a specific programming language

# include the following template

(1) c-format-template
a simple template focusing on vscode environment configuration using codeium .

(2) gcc-project-template
an extremely simple yet modern gcc development environment .

(3) nodejs-format-template
with just these 2 files, you have the ability to automatically check JavaScript and format code .

(4) nodejs-express-template
a simple  web server utilizing the node.js express framework .

(5) nodejs-koa-template
a simple application utilizing the node.js koa framework, which you can consider as a stripped-down version of a tr069 server

(6) nodejs-plus-cpp-template
a simple example of calling c++ code from node.js , you can use this as a foundation to enable your node.js application to utilize C++ code

(7) python-vscode-template
a script used to automatically generate a python runtime virtual environment and configure vscode

(8) vc2019-vscode-template
configure the c compilation and debugging environment for vscode , with vs2019's cl tools
